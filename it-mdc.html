1. Define an analog computer and a digital computer.

Analog computer: A machine that processes continuous data by representing physical quantities (such as temperature or speed) through measurable variables (e.g., voltage).
Example: Speedometer in a car.

Digital computer: A machine that processes discrete data using binary code (0s and 1s), which are interpreted by the computer's central processing unit (CPU).
Example: Laptop or Smartphone.

2. Give an example each of an analog computer and a digital computer.

Analog Computer: Oscilloscope (used to measure voltage signals).
Digital Computer: Desktop computer (used for calculations and data processing).

3. List the main characteristics of the computer.

Speed: Computers can process millions of instructions per second, making them faster than manual calculations.
Accuracy: Computers can perform calculations with high precision.
Automation: Computers can perform tasks automatically without human intervention after being programmed.
Storage: Computers can store vast amounts of data for future use.
Versatility: Computers can be used for a wide range of tasks such as calculations, entertainment, and communication.

4. Give two examples of first-generation computers.

ENIAC (Electronic Numerical Integrator and Computer): Used for ballistic calculations during WWII.
UNIVAC (Universal Automatic Computer): The first commercially successful computer, used for business applications.

5. Give two examples of fifth-generation computers.

IBM Watson: An AI-driven system used for complex problem-solving and data analysis.
Quantum Computers: Computers that use quantum bits (qubits) to perform computations that are significantly faster than classical computers in certain tasks.

6. Define a supercomputer.

A supercomputer is a high-performance computer used for extremely complex calculations, such as weather forecasting, scientific research, and simulations of physical processes.
Example: The Summit supercomputer (USA), which is capable of performing over 200 quadrillion calculations per second.

7. What is the meaning of volatile memory? Also, give an example of volatile memory.

Volatile memory refers to computer memory that loses its data when the power is turned off.
Example: RAM (Random Access Memory) is volatile because it stores temporary data that is erased when the computer is powered down.

8. Define peripheral devices.

Peripheral devices are external devices connected to a computer to enhance its functionality.
Examples: Keyboard, Mouse, Printer, External Hard Drive.

9. Explain in detail the input unit of the computer.

The input unit of a computer allows users to provide data to the system for processing. It consists of devices that convert data from the user into a format that the computer can understand.
Examples:

  • Keyboard: For typing text.
  • Mouse: For pointing and selecting items.
  • Scanner: For digitizing physical documents.

10. What is the purpose of the input interface?

The input interface connects input devices (like keyboard or mouse) to the computer, converting user data into signals that the CPU can process.

11. Explain in detail the output unit of the computer.

The output unit of the computer displays the processed data to the user in a human-readable format. It consists of devices that convert computer data into forms that can be seen, heard, or printed.
Examples:

  • Monitor: Displays graphical output.
  • Printer: Produces hard copies of documents.
  • Speakers: Convert digital sound into audible sound.

12. What is the purpose of the output interface?

The output interface manages communication between the CPU and the output devices. It converts digital data from the CPU into a format that output devices can interpret and present to users (e.g., converting digital signals to analog signals for speakers).

13. Name three pointing devices.

Mouse, Trackpad (Touchpad), Stylus (for touch screens).

14. Describe the features of the mouse.

The mouse is a pointing device that typically has two or three buttons and a scroll wheel. It moves a pointer on the screen to select and interact with items. The mouse can be optical (using light) or mechanical (using a ball).

15. What is the use of a barcode reader?

A barcode reader is used to scan printed barcodes and convert the information into digital data for processing.
Example: Used in supermarkets to read product details and prices.

16. What is the use of a plotter?

A plotter is used for printing vector graphics, including technical drawings, blueprints, and maps, with high precision.
Example: Used by architects to print architectural plans.

17. Define the resolution of the screen.

Screen resolution refers to the number of pixels that can be displayed on the screen, typically represented by width × height (e.g., 1920×1080). A higher resolution means more pixels, resulting in clearer and sharper images.

18. Define the refresh rate of the screen.

Refresh rate is the number of times per second that a screen updates its image, measured in Hertz (Hz). Higher refresh rates result in smoother visuals.
Example: A monitor with a 60Hz refresh rate updates the screen 60 times per second.

19. What is a device driver?

A device driver is software that enables the operating system to communicate with a hardware device, enabling it to function properly.
Example: A printer driver allows a computer to send data to a printer for printing.

20. What is a device controller?

A device controller is a hardware component that manages the data flow between the computer’s CPU and external peripherals, ensuring smooth communication.
Example: A disk controller that manages communication between the CPU and the hard drive.

21. Is a device controller hardware or software?

Hardware

22. Is a device driver hardware or software?

Software

23. Differentiate between Ink-jet printers and Laser printers.

Inkjet Printers: Uses liquid ink and is good for printing images and photos.
Laser Printers: Uses toner powder and is faster, ideal for printing large volumes of text.

24. What is system software?

System software is a set of programs that manage computer hardware and provide a platform for running application software.
Examples: Operating System, Device Drivers.

25. Give two examples of system software.

Windows OS, Linux

26. Describe the two categories of system software.

Operating System: Manages hardware resources and provides an interface for users to interact with the computer.
Utility Software: Performs maintenance tasks like antivirus, disk cleanup, file management tools.

27. What is the need for an operating system?

The operating system (OS) acts as an intermediary between the hardware and software of a computer. It manages hardware resources, runs applications, and provides a user-friendly interface for interaction.

28. Name any three operating systems.

Windows, macOS, Linux

29. What are the uses of system utilities?

System utilities help maintain and optimize system performance. They are used for tasks like file management, system monitoring, and security.
Example: Disk cleanup, antivirus software.

30. List any five system utilities and also list the purpose of each.

Antivirus: Protects against malware.
Disk Cleanup: Removes unnecessary files.
Backup Software: Creates data backups.
Task Manager: Monitors system performance.
Defragmenter: Optimizes file storage.

31. What is the need for programming languages?

Programming languages enable developers to write software that can be executed by computers, defining the tasks and functions to be performed.

32. Name the three categories of programming languages.

Machine Language, Assembly Language, High-Level Language

33. List the key features of machine language.

Binary code: Only uses 0s and 1s.
Fast execution: Directly executed by the CPU.
Difficult to write and debug.

34. List the key features of assembly language.

Mnemonics: Uses readable codes for operations.
Low-level programming: Close to hardware.
Requires an assembler to convert to machine language.

35. List the key features of high-level languages.

Human-readable syntax: Easier to write and understand.
Portable: Programs can run on different platforms.
Requires a compiler or interpreter for execution.

36. What are low-level languages?

Low-level languages are programming languages that are closely related to the hardware, such as machine and assembly languages. These languages provide control over hardware operations but are more difficult to write and debug.

37. Name two high-level languages.

Python, Java

38. What is the use of application software? Give examples.

Application software is used to perform specific tasks like word processing, image editing, or gaming.
Examples: Microsoft Word (word processing), Adobe Photoshop (graphic design).

39. Define: (1) Modem, and (2) NIC.

Modem: A device that converts digital data to analog for internet access.
NIC (Network Interface Card): A hardware component that allows a computer to connect to a network.

40. Define network bandwidth.

Network bandwidth is the maximum rate at which data can be transferred over a network, typically measured in Mbps/Gbps.

41. What is the function of a web server?

A web server hosts websites and serves web pages to users over the internet. It processes requests from clients (browsers) and sends the appropriate data back.

42. What is the use of a web browser?

A web browser is software that allows users to access, view, and navigate websites on the internet.
Example: Google Chrome, Mozilla Firefox.

43. Differentiate between E-mail and Chat.

E-mail: Asynchronous communication where users send messages that may be read later.
Chat: Real-time messaging, often for instant communication between users.

44. What do you mean by guided and unguided transmission media?

Guided transmission media involves physical cables (e.g., fiber optic, coaxial).
Unguided transmission media involves wireless communication (e.g., radio waves, microwaves).

45. Name two guided transmission media.

Twisted Pair Cable, Fiber Optic Cable.

46. How is a coaxial cable different from a twisted pair cable?

Coaxial Cable: Contains a central conductor, insulating layer, shielding, and outer cover, providing better protection against interference.
Twisted Pair Cable: Consists of pairs of wires twisted together, providing less protection from interference compared to coaxial cables.

47. What is the unit of measuring bandwidth?

Bits per second (bps) or megabits per second (Mbps).

48. Define computer network.

A computer network is a system of interconnected computers that share resources and data, allowing communication between devices.

49. Describe briefly the LAN, MAN, and WAN transmission technologies.

LAN (Local Area Network): A network confined to a small geographic area, such as a home or office.
MAN (Metropolitan Area Network): A network that spans a city or a large campus.
WAN (Wide Area Network): A network that covers a large geographic area, like the internet.

50. Name three LAN topologies.

Star, Bus, Ring.

51. Explain briefly how the wireless network works.

A wireless network transmits data using radio waves instead of physical cables. Devices connect to the network using Wi-Fi or Bluetooth signals, allowing for mobility and convenience.
